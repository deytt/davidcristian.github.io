Sobre mim

As tecnologias que estou habituado a utilizar são: Java, Kotlin, JPA com Hibernate, REST com Micronaut / Spark / Spring Framework, Gradle, Maven, JasperReport, Sqlite, Oracle, Git e testes unitários com JUnit.

Meus principais hobbies são ler, programar e jogar. Recomendo muito o livro "A única coisa" do empresário americano Gary Keller. Gosto de aprender coisas novas criando aplicações simples e jogar com os amigos.
